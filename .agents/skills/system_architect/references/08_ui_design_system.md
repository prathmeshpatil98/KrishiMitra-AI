
# KrishiMitra AI Enterprise Design System

## Purpose

This document defines the official UI/UX design system for KrishiMitra AI.

Every page, component, animation, interaction, chart, card, modal, form, and dashboard MUST follow these guidelines.

The UI should feel comparable to:

- OpenAI
- Perplexity
- Linear
- Stripe
- Vercel
- Apple
- Notion

Never generate generic dashboards or Bootstrap-looking interfaces.

This is production SaaS software.

---

# Design Philosophy

KrishiMitra AI should communicate

Trust

Intelligence

Simplicity

Professionalism

Speed

Clarity

The interface should feel calm.

Avoid visual clutter.

# Final Principle

Every generated interface should look like it belongs to a funded AgriTech startup.

When generating UI, always ask:

"Would this design look natural in a product built by Stripe, Linear, OpenAI, or Perplexity?"

If the answer is no,

redesign it before generating.
