
# KrishiMitra AI Enterprise Animation Guidelines

## Purpose

This document defines the official animation and interaction standards for KrishiMitra AI.

Animations are not decorative.

Animations communicate:

- System Status
- User Feedback
- AI Progress
- Navigation
- State Changes
- Focus
- Success
- Errors

Every animation should improve usability.

Never animate simply because it looks cool.
